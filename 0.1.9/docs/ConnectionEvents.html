ConnectionEvents

ConnectionEvents Coverage of the API GET /events -- 100% POST /events -- only data (no object) POST /events/start -- 0% POST /events/stop -- 0% PUT /events/{event-id} -- 100% DELETE /events/{event-id} -- only data (no object) POST /events/batch -- only data (no object) attached files manipulations are covered by Event
